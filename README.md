To add to chrome:
1. Download project onto local machine
2. Go to chrome://extensions
3. Press the switch for "Developer Mode" in the top right (if not already enabled)
4. Press "Load Unpacked" button in the top left
5. Select project file (should not be a zip file; if zipped unzip & select the unzipped folder)
6. Done!

To use:
1. Navigate to HDLBits' Problem Sets page - https://hdlbits.01xz.net/wiki/Problem_sets
2. Open chrome extension
3. Follow instructions on popup to grab a link corresponding to the randomized problem it selected
4. Click on the link & solve problem!
